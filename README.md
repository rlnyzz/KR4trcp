# FastAPI Async Testing Project

Проект демонстрирует асинхронное тестирование FastAPI приложения с использованием `pytest-asyncio`, `httpx.AsyncClient` (через `ASGITransport`) и `Faker` для генерации тестовых данных.

## 📋 Требования

- Python 3.9+
- PostgreSQL (не используется, данные в памяти — для тестов)
- Docker (опционально)
